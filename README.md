# Retranscrire avec Whisper

Support pour l'atelier "Retranscrire ses entretiens avec Whisper" de la BU de Paris 8.

Le dépôt est dérivé du cours pensé par Emilien Schultz intitulé ["python-for-quali"](https://github.com/pyshs/tutorial-quali).

# Python pour les qualitativistes

## Présentation générale

L'objectif est de montrer comment Whisper peut être utilisé pour retranscrire des entretiens en sciences humaines et sociales. La formation donne également quelques clés pour explorer la retranscription en utilisant des éléments de programmation simples en Python.

Elle est couplée avec l'usage du SSP Cloud Datalab de l'Insee : [https://datalab.sspcloud.fr/](https://datalab.sspcloud.fr/).

## Prérequis

Les participants sont donc invités :

1. à créer un compte sur SSP Cloud Datalab avec leur adresse univ-paris8.fr (ou un autre domaine autorisé);
2. à vérifier qu'ils disposent d'un accès aux services Huma-Num en suivant le lien ci-après : [https://humanid.huma-num.fr/](https://humanid.huma-num.fr/)

Il n'y a pas d'autre prérequis pour suivre l'atelier.
